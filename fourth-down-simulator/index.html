Fourth Down Simulator

This beta version uses the current Sportsecon.com port of the canonical SignalPlay fourth-down model, packaged as a simple browser tool for quick decision checks and scenario testing.

Read the starter post Back to all tools

What this beta does

It compares three choices, translates the output into readable guidance, and leaves room for future model expansion, richer assumptions, and cleaner visualization.

The current model is intentionally lightweight. It should be treated as structured decision support, not a finished game-management engine.

Run a fourth-down decision

Uses a 1–99 scale from your own goal line: 1 = own 1, 50 = midfield, 58 = opponent 42, 99 = opponent 1.
